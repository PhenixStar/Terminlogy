Terminolgy has two preset systems: **AI presets** for saving AI provider configurations, and **background presets** for saving custom tab backgrounds.

---

## AI Presets

AI presets let you define multiple AI provider configurations and switch between them from the Terminolgy AI block header dropdown.

### Preset File Location

```bash
tsh editconfig presets/ai.json
```

File path: `~/.config/terminolgy/presets/ai.json`

### Preset Structure

Each preset key starts with `ai@` by convention:

```json
{
  "ai@claude-sonnet": {
    "display:name": "Claude 3 Sonnet",
    "display:order": 1,
    "ai:*": true,
    "ai:apitype": "anthropic",
    "ai:model": "claude-3-5-sonnet-latest",
    "ai:apitoken": "<your anthropic API key>"
  }
}
```

Set a preset as default via `settings.json` or tsh:

```bash
tsh setconfig ai:preset=ai@claude-sonnet
```

### Using a Proxy

Add `ai:proxyurl` to any preset to route AI requests through an HTTP proxy:

```json
{
  "ai@claude-via-proxy": {
    "display:name": "Claude (via Proxy)",
    "ai:*": true,
    "ai:apitype": "anthropic",
    "ai:model": "claude-3-5-sonnet-latest",
    "ai:apitoken": "<key>",
    "ai:proxyurl": "http://proxy.example.com:8080"
  }
}
```

## Background Presets

Custom tab backgrounds can be saved as reusable presets in `~/.config/terminolgy/backgrounds.json`.

```bash
tsh editconfig backgrounds.json
```

Preset keys start with `bg@` by convention. Apply a preset to the current tab:

```bash
tsh setbg bg@ocean-depths
```

Set a preset as the default for all new tabs in `settings.json`:

```json
{ "tab:background": "bg@ocean-depths" }
```

Built-in background presets are defined in the [default backgrounds.json](https://github.com/PhenixStar/terminolgy/blob/main/pkg/wconfig/defaultconfig/backgrounds.json).
